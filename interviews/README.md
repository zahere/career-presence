# Interview Preparation System

> Part of career-presence
> Auto-generates prep materials from master_profile.yaml

## Structure

```
interviews/
├── prep/
│   ├── {company}/           # Company-specific prep (generated)
│   └── template.md          # Base template
├── stories/
│   ├── star_stories.md      # STAR format stories
│   └── talking_points.md    # Key messages
├── questions/
│   ├── behavioral.md
│   ├── technical.md
│   ├── system_design.md
│   └── company_specific.md
└── README.md                # This file
```

## Quick Prep Command

```bash
/interview prep anthropic    # Generate Anthropic-specific prep
/interview stories           # Review STAR stories
/interview practice          # Random question practice
```

## Pre-Interview Checklist

### 24 Hours Before
- [ ] Review company prep doc
- [ ] Practice 30s/2min/5min AgentiCraft pitch
- [ ] Review job description keywords
- [ ] Prepare 3 questions to ask
- [ ] Test video/audio setup

### 1 Hour Before
- [ ] Review STAR stories
- [ ] Quick scan of recent company news
- [ ] Water, notes, quiet space ready
- [ ] Phone on silent

### During Interview
- [ ] Listen fully before answering
- [ ] Use STAR format for behavioral
- [ ] Ask clarifying questions for technical
- [ ] Take notes on interviewer names

### After Interview
- [ ] Send thank you within 24 hours
- [ ] Log in tracking system
- [ ] Note questions asked for future prep
